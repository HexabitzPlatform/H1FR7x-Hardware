# H1FR7x-Hardware

Hexabitz Zigbee Module Hardware Design



Design this module using the STM32G0B1CEU6 MCU and XBRR-24Z8CM-J.

You can find the datasheet and hardware information at the following link:  
https://hub.digi.com/support/products/digi-xbee/digi-xbee-rr/

This design must use two hexagons with a 2-layer PCB stack-up.
